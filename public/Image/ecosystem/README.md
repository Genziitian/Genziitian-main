# Ecosystem page snapshots

Drop the product screenshots here with these exact filenames. The Ecosystem
page (`/ecosystem`) shows them automatically; until a file exists it falls back
to a styled CSS mockup, so the page never looks broken.

| File                    | Screenshot                                   |
|-------------------------|----------------------------------------------|
| `website-desktop.png`   | genziitian.in — desktop homepage             |
| `website-mobile.png`    | genziitian.in — mobile homepage              |
| `dashboard.png`         | class.genziitian.in — student portal (web)   |
| `exam.png`              | lab.genziitian.in — LAB / exam platform (web)|
| `social.png`            | genziitian.live — social platform (optional) |
| `app.png`               | Mobile app home screen                        |

Recommended: PNG/JPG, web shots ~1600×1000, mobile/app shots ~750×1500.
